DCS Library service can be accessed via port {{ .Values.servicePort | default "10810" }} on the following DNS name from within your cluster:
{{ template "library-svc.fullname" . }}.{{ .Release.Namespace }}.svc.cluster.local

To probe the liveness of the service:

1. Run an Ubuntu pod that you can use as a client:

    kubectl run -i --tty ubuntu --image=ubuntu:16.04 --restart=Never -- bash -il

2. Install the curl:

    $ apt-get update && apt install curl -y

3. Connect to service:
    $ curl -XGET {{ template "library-svc.fullname" . }}
